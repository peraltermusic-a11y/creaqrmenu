Questo QR Code rimarrà sempre lo stesso. Stampalo e mettilo sui tavoli.

Come cambiare il link del menu:

  1. Vai nella sezione Issues del tuo repository GitHub.
  2. Apri la prima Issue (quella contrassegnata con il numero 1).
  3. Modifica il Titolo della Issue inserendo il nuovo URL del tuo menu (es. https://tuosito.com/foto-menu.jpg).
  4. Salva. Il QR Code si aggiornerà istantaneamente per tutti i clienti!
